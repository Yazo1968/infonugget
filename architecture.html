How It Should Work

The browser only handles what you see. Everything else happens on the server — reliably, atomically, without depending on your browser staying open.

Your Browser
Only does:
Shows buttons & panels
Captures your clicks
Displays images & text
Sends requests to API
Shows loading spinners
Clean • Fast • Simple
No more prompt building
No more image conversion
No more auto-save timers
No more retry logic
No more serialization
One Request
One Result

generate-card
